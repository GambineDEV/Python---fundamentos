Repositório dedicado ao aprendizado dos conceitos fundamentais da linguagem **Python**. Aqui estão reunidos os meus primeiros scripts, exercícios de lógica e testes práticos estruturados para consolidar a base da programação orientada a objetos, manipulação de dados e lógica condicional.

---

## 📁 Estrutura do Repositório

O repositório contém arquivos de script (`.py`) e notebooks de testes estruturados para estudar os seguintes tópicos:

* **Variáveis e Tipos de Dados:** Manipulação de strings, inteiros, floats e booleanos.
* **Estruturas de Condição:** Tomada de decisão no código utilizando `if`, `elif` e `else`.
* **Estruturas de Repetição:** Criação de loops utilizando `while` e `for` para varredura de dados.
* **Coleções de Dados:** Trabalho prático com listas, tuplas, dicionários e matrizes (listas de listas).
* **Funções e Modularização:** Criação de funções personalizadas, passagem de parâmetros e retorno de valores.

---

## 🛠️ Tecnologias e Ferramentas

* **Python 3:** Linguagem principal utilizada para o desenvolvimento de todos os scripts.
* **VS Code / Jupyter Notebooks:** Ambientes utilizados para codificação, testes rápidos e depuração de erros.
